Facebook SDK for Python

Unofficial Python Facebook SDK

View project onGitHub

This client library is designed to support the Facebook Authentication and Graph API. Read more about the Graph API

Example of use:

    request  = {'signed_request':'...'}
    fb       = Facebook(request,app_id,app_secret)
    session  = fb.getSession()
    graph    = fb.getGraph()
    user     = graph.get_object("me")
    friends  = graph.get_connections(user["id"], "friends")
